# 💰 Sentimento de Mercado B3

> Análise de sentimento de notícias e redes sociais sobre ações da B3 em tempo real, gerando sinais de trading e alertas de movimento.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/sentimento-mercado-b3/

## 📊 Resultados-chave (KPIs)
**0.91** F1 PT-BR · **380** Tickers · **1.240** Sinais/dia · **73%** Acerto

## 🧠 Tecnologias de IA
NLP · Sentiment Analysis · LLM · BERTimbau

## ✨ O que faz
- Análise PT-BR fine-tuned (BERTimbau)
- Score de sentimento por ticker
- Correlação com preço
- Alertas de mudança brusca

## 🗄️ Fonte de dados real
[Notícias financeiras + B3](https://www.b3.com.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com Notícias financeiras + B3
2. **Processamento com IA** — NLP · Sentiment Analysis · LLM · BERTimbau
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
